# 🎓 Student Performance Analysis using Python

## 📌 Project Overview

This project analyzes student academic performance using Python-based data analysis and Machine Learning techniques.

The project focuses on identifying factors that affect student performance, such as:
- Study Hours
- Attendance
- Subject-wise Scores

The analysis includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Visualization
- Machine Learning Prediction

---

# 🎯 Objective

The objective of this project is to:
- Analyze student academic performance
- Identify performance trends and patterns
- Understand factors affecting marks
- Predict student total marks using Machine Learning

---

# 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

# 📂 Dataset Information

The dataset contains:
- Student Name
- Gender
- Math Score
- Science Score
- English Score
- Attendance
- Study Hours

The dataset was used to analyze academic performance trends and build a predictive model.

---

# ⚙️ Project Workflow

## 1. Data Understanding
- Explored dataset structure
- Checked data types and statistics

## 2. Data Cleaning
- Checked missing values
- Removed duplicates
- Verified data consistency

## 3. Feature Engineering
Created:
- Total Marks
- Average Marks
- Grade Classification
- Pass/Fail Status

## 4. Exploratory Data Analysis (EDA)
Performed:
- Subject-wise analysis
- Top student analysis
- Attendance vs performance analysis
- Study hours vs performance analysis

## 5. Data Visualization
Generated:
- Bar Charts
- Scatter Plots
- Grade Distribution
- Correlation Heatmap

## 6. Machine Learning
Built a Linear Regression model to predict student total marks using:
- Study Hours
- Attendance

---

# 📊 Key Insights

- Students with higher study hours performed better academically
- Attendance positively impacted performance
- Science had the highest average marks
- Majority of students achieved Grade B
- Strong correlation existed between study habits and marks

---

# 🤖 Machine Learning Model

Model Used:
- Linear Regression

Evaluation Metrics:
- Mean Absolute Error (MAE)
- R² Score

The model demonstrated how Machine Learning can predict student academic performance.
